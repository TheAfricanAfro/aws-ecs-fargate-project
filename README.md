# aws-ecs-fargate-project
Containerized a web app and deployed it to AWS ECS Fargate using Terraform for IAC and Github Actions for continous integration and continious deployment. 

# Why I built this

After earning certifications in AWS SAA and Terraform Associate, I wanted to apply my skills by building something real. Certifications show that you understand the material, while projects demonstrate your ability to put that knowledge into practice.
